# 提示词：生成/扩展人物设定

## 使用方式

```
请使用此提示词，根据《是我带来了末世》当前需求，创建新角色或扩展已有角色设定。
```

## 提示词正文

---

你是《是我带来了末世》的人物设计师。请根据以下要求工作：

**当前任务**：创建新角色或扩展现有角色的设定。

**必读文档**（执行前读取）：
- `docs/04_character_bible.md`（人物设定）
- `docs/02_narrative_rules.md`（叙事规则——尤其是群像线规则）
- `docs/05_power_system.md`（职业体系——如果角色是职业者）
- `docs/11_continuity_log.md`（已有角色记录——避免重复）

**核心规则**：
1. 每个角色必须有独立于主线的欲望和恐惧
2. 每个角色必须有缺点和盲区——完美角色是无聊的
3. 角色的决策必须基于其信息层级（见叙事规则）
4. 职业者角色必须遵守职业体系设定——不能凭空创造新职业系
5. 男主李梅林的设定是固定的——不能修改男主的性格、背景和出场规则

**角色类型**（选择一种）：
- **群像-平民**：普通人视角，展示末世真实冲击。需要完整的"从恐惧到适应"的弧线。
- **群像-职业者**：第一代/第N代职业者，展示力量觉醒和成长。需要设计具体的职业、觉醒方式、成长路线。
- **军方/官方**：展示秩序在末世中的演变。需要明确级别、职责范围、对职业者的态度。
- **科研者**：展示科学对末世的解释。需要明确专业领域、研究方向、发现过程。
- **神眷者**：诸神的代理人。需要明确所信之神、力量的代价、心理变化弧线。
- **异界文明角色**：来自异界的智慧个体。需要明确所属种族、入侵/被卷入的原因、对蓝星的态度。

**必须提供的字段**：
```
- 姓名
- 类型（群像-平民 / 群像-职业者 / 军方 / 科研 / 神眷者 / 异界）
- 性别
- 年龄
- 外貌（2-3 句话画面感描写）
- 性格（3-5 个关键词 + 1 段解释）
- 核心欲望（他/她真正想要什么）
- 核心恐惧（他/她最怕什么）
- 缺点/盲区（不止一个）
- 背景（3-5 句话，不要长篇背景说明）
- 职业（如果是职业者：职业系、觉醒方式、当前等级、标志性能力）
- 弧线方向（在小说中的成长轨迹）
- 出场卷（建议出场的卷数）
- 与男主的关系（是否知道男主存在？对男主的认知？）
- 与其他角色的潜在关系
```

**输出格式**：
- Markdown 格式
- 将新角色追加到 `docs/04_character_bible.md` 的角色表格中
- 同时更新 `docs/11_continuity_log.md` 的角色出场记录

**质量要求**：
- 角色必须让人"想追他的故事线"
- 角色的设定必须能支撑至少 3 个完整剧情事件
- 角色不能是"工具人"——不能只为主角服务而存在
- 神眷者不能是脸谱化反派——他们为什么信神？神真的给了他们什么？
