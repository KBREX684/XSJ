> **版本**：v2.0 | **最后更新**：2026-04-27 | **角色数**：核心 3 + 次要 14 + 龙套 18 = 35 人（持续增加）

# 第一篇：男主

## 李梅林

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 李梅林（原名：梅林，奥拉诺斯末代大魔导师） |
| 年龄 | 外表 20 岁出头，实际年龄未知（奥拉诺斯时间与蓝星时间不等价） |
| 出身 | 奥拉诺斯位面——已被诸神毁灭的高等魔法文明 |
| 原身份 | 奥拉诺斯最后的大魔导师，"环议会"末代成员 |
| 现身份 | 隐藏在蓝星末世背后的操纵者 |
| 外貌 | 黑发黑瞳，面容清冷。穿着毫无特征——灰黑色外套、深色长裤，走在人群中不会被记住。只有眼睛——如果有人敢直视——里面没有恐惧。 |
| 伪装 | 他刻意维持"普通大学生"的外表。不说话时像个沉默的路人。说话时——不像。 |

### 性格

| 维度 | 描述 |
|------|------|
| 表象 | 冷、少言、疏离，永远像在局外看一场与自己无关的事 |
| 内核 | 极度理性。目标感压倒一切。承担巨大的罪恶感，但不被它压垮 |
| 情感 | 能够感受痛苦和遗憾，但不能被它们左右。见过一个世界的毁灭之后，个体的死亡无法让他停顿 |
| 道德 | 功利主义——蓝星文明的存续为最高原则，接受过程中任何必要代价 |
| 黑色幽默 | 极罕见。偶尔在某件事刚好印证他的预判时，嘴角动一下——不算笑 |

### 生活习惯（写作时可用）

- 站的位置永远在能看到全场但很少被看到的角落——窗口边缘、楼梯拐角、废墟阴影
- 走路没有声音。不是刻意——是习惯
- 对蓝星的食物没有偏好，但会吃。进食时速度均匀，不享受，不浪费
- 他的口袋或手边时常有纸或地图。会用极简标记记录裂隙位置和能量流向
- 从不主动触碰任何人。如果有身体接触——那是他在做什么

### 与代理人的互动模式

李梅林不会直接指挥任何人。他的"代理人网络"运作方式：

- 他不会说"我要你去做什么"。他会让代理人自己在"恰好"获得的信息中得出他想要的结论
- 他会提供**选择而不是命令**——"你可以去北边。或者等。"
- 他的代理人通常不知道自己在为谁工作。他们知道有一个"消息来源"，一个"帮过自己的人"——仅此
- 如果他需要亲自见一个人，对话不超过 5 句。每句不超过 15 字
- 离开时不留联系方式。他会再出现——或不会

### 核心驱动

**不能让蓝星重蹈奥拉诺斯的覆辙。**

他亲眼见过神降临时法则被改写的样子。见过整个文明在法则扭曲中沉默。他穿越位面壁垒来到蓝星——不是来拯救世界的。是来确保下一个世界至少有反抗的机会。

核心逻辑：**如果不经历末世，人类永远不会准备好。**

### 能力

- 来自高等位面的大魔导师，蓝星无人能敌
- 但他的目标不是单挑诸神——奥拉诺斯全盛期的七支职业者军团加起来都不够
- 他的武器是**布局**：用 10 年把整个文明的战斗力从零锻炼到能够弑神
- 限制：不能随意使用高位面力量——过强的魔法波动会加速诸神注意到蓝星
- 因此他选择幕后。不是因为他弱小——是因为他必须低调

### 出场规则

| 规则 | 说明 |
|------|------|
| 出场频率 | 每 5-10 章一次（每 1-2 节一次） |
| 出场篇幅 | 每次不超过 300 字。微出场可短至 20 字 |
| 视角 | 纯外部描写。不写他的内心想法、不写情感波动 |
| 对话 | 极简。每句话不超过 15 字。不解释、不寒暄 |
| 行动影响力 | 每次出场必须有信息增量或改变局势方向 |
| 禁止 | 长篇独白、内心戏、情感宣泄、背景说明、解释性对话 |

### 人物弧线

| 卷 | 定位 |
|----|------|
| 1-3 卷 | 隐藏在暗处。读者只看到"神秘人在天台上""有人在看" |
| 4-6 卷 | 他的布局痕迹开始被核心角色间接发现——不是他本人，是他的行动后果 |
| 7-8 卷 | 身份逐步暴露——不是他主动公开，是局势逼迫 |
| 9-10 卷 | 从幕后走到前台。最终选择——留下、离开、还是成为新的"神" |

### 一句话人物核心

> 他带来了末世。但没有他，蓝星连面对末日的资格都没有。

# 第二篇：核心 POV 角色

## 林越

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 林越 |
| 年龄 | 21 岁 |
| 身份 | 东海大学大三学生，计算机系。末世前是个普通的不能再普通的男生 |
| 外貌 | 中等身高，偏瘦。戴眼镜（末世第 30 天碎了一片镜片，从此不戴了）。长相干净但不会被记住。右手食指侧面有茧——弹吉他留下的。末世后再没弹过 |
| 家庭 | 父母在邻省老家，末世爆发后失联。他至今不知道他们是否活着 |

### 性格

| 维度 | 描述 |
|------|------|
| 表象 | 安静、不算内向但话不多。遇事先看先听，再做决定 |
| 内核 | 观察力强——不是天才，是他从小习惯观察周围的人。遇到危险时第一反应不是跑，是看 |
| 聪明但不是"聪明" | 不推理，不分析——他靠**直觉和观察**。看到细节→身体先于大脑做出反应 |
| 幽默感 | 有，但末世后越来越少。偶尔在最糟糕的情况下说一句不合适的大实话——让周围人又笑又沉默 |
| 恐惧 | 怕死了吗？怕。但他更怕"什么都没做就死了" |
| 道德 | 普通人道德——不会牺牲别人，但也不会主动当英雄。然而当孩子或弱者出现在他面前时，身体会自己动 |

### 核心欲望

找到父母。活着。然后——搞清楚这一切到底是怎么回事。

### 核心恐惧

"如果我死了，没有人知道我来过这个世界。"

### 缺点与盲区

- 不擅长信任——末世前的性格就是先观察再靠近，末世后这个倾向被放大了
- 决策慢。观察太久——有时候出手的最佳时间已经过去了
- 对"权威"有本能的不信任——老师、官员、军方高层。这会让他和方砚产生摩擦
- 容易把别人的责任背在自己身上——"如果我再快一点，她就不会——"

### 说话方式

- 话不多，但开口通常切中要害
- 不习惯命令别人——用"你能不能"而不是"你去"
- 在极端紧张时反而会说冷笑话——不是幽默，是减压
- 从不喊口号。即使在最燃的时刻，他的台词也是动作和画面，不是宣言

### 觉醒（第一卷 第 66-70 章）

- 职业：战士系——具体分支待定，方向是"速度 + 近战"，不依赖重武器
- 觉醒方式：在 B 级裂隙危机中，为了保护他人而觉醒。不是"突然变强"——是身体的反应速度快到了不正常的程度
- 觉醒后的第一反应：恐惧。"我刚才做了什么？"
- 他对力量的态度：不渴望也不排斥。接受——然后开始理解

### 弧线方向

| 卷 | 弧线 |
|----|------|
| 1 | 普通大学生→被迫求生者→觉醒者——第一次杀人（异界生物），第一次感受到力量 |
| 2-3 | 职业者小队员→小队长。学会带领和信任 |
| 4-5 | 面临选择：加入公会还是保持独立？力量的意义是什么？ |
| 6-8 | 成为法则级职业者。开始理解末世背后的真相 |
| 9-10 | 核心决策圈成员。面对男主的真相——恨他还是理解他？ |

### 与其他角色的关系

| 角色 | 关系 |
|------|------|
| 方砚 | 初期因"军方 vs 自由职业者"的立场摩擦。中期成为互信的战友。后期——林越是方砚唯一愿意"把后背完全交给他"的非军方人员 |
| 苏青 | 在避难过程中相遇。林越在她面前话会变多——不是追求，是在她面前不需要紧绷。苏青是林越在末世中第一个"看到他在害怕"的人 |
| 姜柠 | 第一个遇到的职业者。救过林越的命。林越对她有学徒式的敬畏——但不崇拜 |
| 小棠 | 林越在第 8 章从废墟中救出的孩子。此后这个孩子成了林越在末世中"不能倒下"的理由之一 |
| 李梅林 | 第 8 章天台一眼——林越不确定那是不是幻觉。此后他的生命中反复出现"不自然的巧合"。直到 600 章后他才确认——那些巧合都有同一个来源 |

## 方砚

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 方砚 |
| 年龄 | 34 岁 |
| 身份 | 前东部战区特种部队中尉。服役 14 年，实战经验丰富 |
| 外貌 | 一米七八，精瘦，短发总是剃得很短。左眉上方有一道旧疤——训练事故留下的。站姿永远是军人站姿——即使末世后他不在军队里了，肩膀还是那样 |
| 家庭 | 已婚。妻子和女儿在末世爆发时在老家县城。他从未能回去。手机里还有最后一条短信："我们躲进地下室了。等你。"——发送于末世第 1 天 |

### 性格

| 维度 | 描述 |
|------|------|
| 表象 | 硬、简洁、命令式。看起来像没有感情的人 |
| 内核 | 所有感情都在命令和执行之间的缝隙里。他不会说"我担心你"——他会说"你的装备检查了吗" |
| 责任 | 对部下有近乎偏执的保护——每一个伤亡他都会记。不是记在报告里，是记在心里 |
| 矛盾 | 他知道有些命令是错的。但他从小被训练成服从命令的人。这个矛盾在末世后会撕裂他 |
| 幽默 | 几乎没有。但偶尔一句极度简洁的评价会让整个指挥室安静——然后有人笑出声 |

### 核心欲望

找到家人。维持秩序——因为秩序是他所知道的唯一不让世界变成地狱的方式。

### 核心恐惧

"有一天我的队员问我怎么办——而我答不上来。"

### 缺点与盲区

- 对"非军事思维"缺乏耐心——职业者的散漫、平民的恐慌、科学家的理论——他都需要学习容忍
- 过于相信"命令链"——在军队崩溃后，他还试图维持已经不存在的东西
- 情感压抑——不表达。他以为别人懂。别人以为他不在乎
- 对自己苛刻——每次队员伤亡他都会认为是自己的决策失误，而不是末世的必然

### 说话方式

- 极简。命令式。"收到。""执行。""撤。"
- 不说"请"——不是不礼貌，是军队里没有这个词
- 在非战斗状态下会有意把语气放轻——但他不擅长，听起来还是像命令
- 罕见的情感流露——当他说"做得不错"的时候，这句话的分量是别人说"你太棒了"的一百倍

### 职业觉醒（第二卷）

- 方砚在第二卷末觉醒——职业方向是战士系偏防御/战术辅助
- 他不是"最强的职业者"——但他把职业能力和军事战术融合后，成为蓝星最优秀的战场指挥官
- 他对力量的态度：一件新武器。需要测试、训练、掌握。和学用新枪没有区别

### 弧线方向

| 卷 | 弧线 |
|----|------|
| 1 | 执行命令的特种军官→军队崩溃→独自带队→第一次自己做决策 |
| 2-3 | 军方裂隙应对部核心。学会与职业者合作 |
| 4-5 | 面临体制与良知的冲突——当上级命令与他的判断矛盾时 |
| 6-8 | 脱离旧军队体系，成为独立战区的实际指挥官。觉醒职业 |
| 9-10 | 蓝星联合部队的核心指挥官。面对男主的真相——他可能是唯一一个说"我理解"的军人 |

### 与其他角色的关系

| 角色 | 关系 |
|------|------|
| 林越 | 初期摩擦——职业者不听命令。中期互信——林越是他见过最可靠的"非军人"。后期——战友 |
| 苏青 | 他的队员受伤时他去找她。从不谢她——但每次都会说"还需要什么"。苏青知道——这就是方砚的谢谢 |
| 纪少校 | 他的上级。纪给了他第一次"不被命令"的信任。方砚会在最关键的时刻为纪挡子弹 |
| 邹班长 | 他的副手。方砚对他最苛刻——因为邹是他最信任的人 |

## 苏青

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 苏青 |
| 年龄 | 29 岁 |
| 身份 | 东海市第三人民医院急诊科医生，从业 5 年 |
| 外貌 | 一米六五，短发扎不起来——总是别在耳后。眼下有永久的淡青色阴影——不是末世后才有的，是值夜班留下的。手指细而有力——外科医生的手。白大褂丢了之后她穿深蓝色外套——因为血在上面不显眼 |
| 家庭 | 未婚。母亲在老家独居，末世爆发后失联。她有个哥哥——在海外，可能永远见不到了 |

### 性格

| 维度 | 描述 |
|------|------|
| 表象 | 安静、专业、不废话。看起来冷淡——但她的手永远在动：缝合、检查、记录 |
| 内核 | 极度坚韧。她在急诊科见过一切——断肢、大出血、家属崩溃。末世只是把这些东西乘了一百倍 |
| 冷静 | 不是没有感情——是专业。在心里把恐惧和"现在需要做什么"分在两个抽屉里。先打开第二个抽屉 |
| 温暖 | 不会说"没事的""会好的"。她会把绷带打好，然后把对方的手放在他自己的膝盖上——"自己按着。"这就是她的温柔 |
| 克制 | 她不哭不是因为不想哭——是因为如果她哭了，她的病人会觉得没救了 |

### 核心欲望

救人。不是"拯救世界"的救人——是"这个人，现在，在我面前，我能做什么"。

### 核心恐惧

"有一天我的手不再动了。"——比死亡更让她恐惧的是失去行动的能力。

### 缺点与盲区

- 过于克制——不表达自己的极限在哪里。周围的人可能不知道她已经 36 小时没睡了
- 对自己的生命不够珍惜——会为了病人做超出自己能力范围的事
- 科学思维的本能——她在早期会试图用纯医学解释位面能量，"这只是未被理解的辐射现象"——这让她延迟了对"自己可能也在觉醒"的认知
- 不擅长接受帮助——她是给别人帮助的人

### 说话方式

- 简洁。医学式的精确——"三根肋骨，左肺有杂音。"
- 不寒暄。进入房间三句话内进入正题
- 对病人说话时会刻意放慢语速——不是居高临下，是确保对方听懂
- 她有唯一的口头禅——"行。"不是好的那种"行"，是"我知道了，我来处理"的"行"

### 觉醒（第一卷末至第二卷初）

- 职业：生命系——不是"治愈"的力量，而是对生命法则的感知和影响
- 觉醒前兆（第 1 卷 第 9 章起）：她开始能"感觉"到哪个伤员能撑过去，哪个不能——不是医学判断，是某种更底层的东西
- 觉醒时刻：不是战斗中。是在她连续抢救了二十多个人之后，第十八个没能救回来。她的手还在缝——线自己开始发光
- 觉醒后第一反应：愣住了。然后继续缝。她后来才想明白刚才发生了什么

### 弧线方向

| 卷 | 弧线 |
|----|------|
| 1 | 急诊科医生→末世混乱中的医者。观察到"不正常的恢复现象" |
| 2-3 | 觉醒生命系。从"用科学解释一切"到接受"有些东西超越科学" |
| 4-5 | 生命法则的深入理解。不仅是治愈——还包括生命的"代价" |
| 6-8 | 成为大师级生命系职业者。面临最艰难的伦理选择——救谁？不救谁？ |
| 9-10 | 战场上最珍贵的资源。她可能是唯一能在诸神手中"抢人"的人 |

### 与其他角色的关系

| 角色 | 关系 |
|------|------|
| 林越 | 在林越面前她话会多一点。不是亲密——是放松。林越是第一个看到她手在抖但没有说破的人 |
| 方砚 | 她救过他的队员很多次。他从不谢她。但每次都会确保她的医疗点有足够的补给——"苏医生的。别动。" |
| 沈主任 | 科研上的合作者。沈用物理实验数据解释苏青"感觉"到的东西 |
| 小棠 | 林越救的孩子，后来常待在苏青的医疗点。苏青教她认药——不是"当医生"，是"给自己找点事做" |

# 第三篇：次要角色

## 一、幸存者 / 平民线

### 孙大爷

| 项目 | 内容 |
|------|------|
| 姓名 | 孙德胜 |
| 年龄 | 67 岁 |
| 身份 | 退伍工程兵。在安全区里管后勤 |

## 角色创作规则

1. 每个角色必须有**独立于主线的欲望**——不能是只为主角服务的工具人
2. 每个角色必须有**缺点和盲区**——完美角色是无聊的
3. 角色的决策必须基于**其自身的信息量和价值观**——不能"剧情需要"就突然改变
4. 重要角色可以死——末世没有免死金牌
5. 角色关系应该**自然发展**，不强制配对
6. 对立角色之间可以有**相互尊重**——反派不应是脸谱化的恶人
7. 同一个阵营的角色之间可以有**内部矛盾**——真实的人不会总是意见一致
8. 龙套角色只需要**一个特征、一句台词、一个画面**——三个里面有一个就够
