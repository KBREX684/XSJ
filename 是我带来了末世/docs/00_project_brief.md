## 二、一句话卖点

**末世是他亲手带来的——但这恰恰是他拯救世界的唯一方式。**

---

## 三、核心命题

末世确实是男主带来的。但没有这场末世，蓝星文明在面对诸神时连反抗的资格都没有。

这不是"穿越者在末世中求生"——这是"亲手制造末世以拯救世界"。全书 900 章都在回答一个问题：**如果你知道一种最残酷的手段是唯一的拯救，你做不做？**

---

## 四、男主李梅林

原名梅林。来自已被诸神毁灭的高等魔法文明奥拉诺斯——他是那个世界最后的大魔导师。穿越到蓝星后化名李梅林，用五年时间暗中布局，然后主动打开了位面通道。

- **定位**：幕后操纵者。蓝星无人能敌，但他不单挑——他布局
- **出场**：每 5-10 章一次，每次不超过 300 字。只写他做什么，不写他想什么
- **说话**：每句话不超过 15 个字。不解释、不寒暄
- **核心张力**：读者知道他在操纵一切，书中角色不知道；读者不知道他的全部动机，他在更高维度

## 六、世界观三支柱

1. **多位面宇宙**：神域→高等位面（奥拉诺斯）→中级位面（蓝星）→低等位面（裂隙连接）。诸神是位面收割者，正在评估蓝星
2. **职业体系**：不是系统/游戏——来自奥拉诺斯的法则工程学。灵魂与法则融合→觉醒→进化。六大职业系，七级无数字等级
3. **末世本质**：不是天灾——是梅林人为打开的位面通道导致的能量污染。裂隙是位面壁垒上的裂缝

## 七、文风铁律

四字诀：**冷、紧、画、藏**。

- **冷**——不煽情。不替读者感动。用画面让读者自己动容
- **紧**——砍一切无关描写。每句话要么推剧情，要么画画面
- **画**——让读者"看见"。写"他把杯子捏碎了"不写"他很愤怒"
- **藏**——男主不写内心。诸神前 200 章不现身。真相分七层阶梯逐渐揭示

## 八、差异化特征

| 不是 | 是 |
|------|-----|
| 不是丧尸末世 | 位面裂隙、异界生物、能量污染——非单一怪物类型 |
| 不是系统文 | 职业觉醒——灵魂与法则的融合。无面板、无数值、无提示音 |
| 不是全民转职 | 力量是挣来的——理解、顿悟、代价。不是点天赋树 |

| 不是普通幕后流 | 男主就是末世的制造者——这个秘密贯穿全书 900 章 |

## 十一、创作工作流

```
卷大纲(08) → 节纲(09) → 章纲(10) → 写正文(novelist) → 自检(13) → 编辑审查(editor) → 连续性检查(continuity-checker) → 更新记录(11) → 入库(final/)
```

1. 选卷：读 `08_volume_outline.md` 确认本卷核心事件
2. 选节：读 `09_section_outline.md` 确认本节 5 章蓝图
3. 定章：读 `10_chapter_outline.md` 确认单章钩子/爽点/POV
4. 写作：切换 novelist agent → 写出 `drafts/volume_XX/ch_XXXX.md`
5. 自检：对照 `13_quality_checklist.md` 逐条检查
6. 审查：切换 editor agent → 审节奏/文风/AI 腔
7. 连续：切换 continuity-checker → 更新 `11_continuity_log.md`
8. 入库：通过审查后移入 `final/volume_XX/ch_XXXX.md`
