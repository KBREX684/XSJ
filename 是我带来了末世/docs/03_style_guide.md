# 03 — 文风设定

> 本文档是《是我带来了末世》文风的唯一标准。所有章节的写作风格必须严格遵守。
>
> **版本**：v2.0 | **最后更新**：2026-04-27

---

## 一、整体文风

### 1.1 核心定义

**冷峻克制，笔锋如刀。**

> 不煽情、不废话、不铺陈。每句话要么推剧情，要么画画面。让读者"看见"，而不是"被告知"。

### 1.2 总原则

| 原则 | 说明 | 错误示范 | 正确示范 |
|------|------|---------|---------|
| **画面感优先** | 用动作和细节传递情绪，不写抽象形容词 | "他非常愤怒。" | "他把杯子捏碎了。" |
| **信息密度高** | 每段至少含剧情推进/人物塑造/世界观展示之一 | "天色渐渐暗了下来，夜幕降临了。" | 直接删掉——除非黑暗本身是威胁 |
| **节奏紧凑** | 无关描写全部砍掉。过渡直接切 | "他穿过三条街，经过两栋废墟，终于到了。" | 下一段直接"方砚推开门。" |
| **对话自然** | 角色说人话，不做舞台剧对白 | "如你所知，裂隙分为六个等级……" | "你知道这东西分几级吗？" |
| **冷静叙述** | 不替读者感动。用画面让读者自己感动 | "这一刻，所有人都被深深震撼了。" | "没有人说话。方砚数了一下——十二个人，没有一个在动。" |

### 1.3 句式与节奏

- **短句为主**（5-15字），中句为辅（15-30字），长句极少（仅用于节奏变化）
- **一段不超过 5 句话**（对话段除外）
- **不堆砌形容词副词**——砍掉"非常""十分""极其""令人震撼的""不可思议的"
- **无对话心理描写禁止超过 100 字**（男主禁止任何心理描写）
- **叙述密度**：每 200 字内至少有 1 个具体的感官细节

### 1.4 最小改动对照（写作自检利器）

以下每组对照展示了"可以接受的写法"和"更好的写法"：

| # | 普通写法 | 优化写法 | 改动要点 |
|---|---------|---------|---------|
| 1 | "他感到一阵恐惧从心底升起。" | "他的手开始抖。他从不知道自己的手能抖成这样。" | 外部动作 > 内部宣告 |
| 2 | "这座城市已经被彻底摧毁了。" | "他站在曾是十字路口的地方。四条街，每条街的尽头都是天空。" | 具体细节 > 抽象概括 |
| 3 | "她的眼神中充满了绝望。" | "她没有眨眼。不是不想——是忘了。" | 反常细节 > 情绪标签 |
| 4 | "空气中弥漫着令人不安的气氛。" | 直接写让人不安的具体事物——一束蓝紫色的光、一个停摆的钟、一句无人说的话 | 写来源 > 写感受 |
| 5 | "他似乎想要说什么，但最终还是没有开口。" | "他的嘴张开又合上了。" | 精确动作 > 模糊推测 |
| 6 | "这一刻，他意识到自己可能再也回不去了。" | 删除——如果前面写得好，读者已经意识到了 | 省略 > 点破 |

## 二、分线文风

### 2.1 群像线文风

**关键词：沉浸、真实、有血肉**

#### 群像文风正例

```
林越贴着墙根蹲下，把呼吸压到最低。头顶上有什么东西爬过去，带起一阵腥臭的风。

他数了十二只脚——不，十四只。每条腿有成人手臂那么粗，关节处挂着不知道是什么的碎肉。

他的手在抖。不是因为害怕——他告诉自己。是因为冷。初秋的夜里不该这么冷。

它停下来了。

林越的心也停了。
```

#### 群像文风反例（同场景的错误写法）

```
林越感到非常恐惧。他蹲在墙边，尽量不发出声音。头顶上有一只巨大的异界生物正在爬过，它看起来十分可怕。此时此刻，林越意识到自己正处于生死攸关的危险之中。
```

**反例诊断**：告诉读者"恐惧"和"可怕"而不是让读者感受。没有具体感官。没有节奏变化。

#### 群像线常见错误

| 错误 | 诊断 | 修正 |
|------|------|------|
| 把角色写成"摄像机"——只记录看到什么，没有感受 | 群像线的力量在于角色的主观体验 | 每段描写后跟一句角色的生理或心理微反应 |
| 内心独白 > 100 字 | 读者会跳过 | 砍到 100 字以内。用动作把心理"翻译"出来 |
| 灾难描写过于"干净"——废墟像布景 | 末世必须有腐烂、灰尘、气味 | 每写一个场景，至少加入 2 种感官（视觉 + 嗅觉/听觉/触觉） |
| 角色"太清醒"——恐惧中的决策过于理性 | 人在极端恐惧中的决策通常是冲动的、错误的 | 让角色在恐惧中做出不完美的选择 |

### 2.2 男主线文风

**关键词：极简、压迫、信息密度极高**

#### 男主文风正例

```
李梅林站在天台边缘。

他脚下是燃烧的城市。

他看了片刻。

"裂隙比预计多了一个。"

他转身，空气在他身后闭合。天台上什么痕迹都没有留下。
```

#### 男主文风反例

```
李梅林站在天台的边缘，俯视着脚下正在燃烧的城市。他的目光扫过一片片废墟，心中分析着当前的局势。按照他的计算，本应只有七个裂隙，但下方城市中出现了第八个——这意味着计划出现了偏差。他沉思片刻，自言自语道："裂隙比预计多了一个。"然后他转身离开，使用了空间折叠的能力让自己消失在了原地。
```

**反例诊断**：内心活动（"心中分析""沉思"）、冗余修饰（"俯视着""一片片""扫过"）、能力说明（"使用了空间折叠的能力"）——全部破坏男主的极简压迫感。

#### 男主描写的"必须有"和"绝不能有"

| 必须有 | 绝不能有 |
|--------|---------|
| 他在做什么（外部动作） | 他在想什么（内心活动） |
| 他说了什么（≤15 字） | 他为什么这么说（解释） |
| 环境对他行动的反馈——"空气在他身后闭合" | 他的情绪——"他感到愤怒/悲伤/犹豫" |
| 他出现和消失的干净画面 | 他使用能力的过程说明 |
| 让读者猜 | 告诉读者答案 |

#### 男主出场的最小单元

只需三样东西就构成一次有效的男主出场：
1. **位置**：他在哪？（天台/废墟边缘/人群之外/某个角落）
2. **动作**：他做了什么？（看了/说了/走了/合上了地图/捏碎了一个东西）
3. **消失**：他怎么离开的？（转身走/空气闭合/什么都没留下）

缺一不可。缺一就变成了"男主路过了"。

#### 男主极简对话库（可供参考的语气模型）

| 场景 | 李梅林可能说的话 | 字数 |
|------|-------------|------|
| 发现计划偏差 | "多了一个。" | 4 |
| 确认某事完成 | "够了。" | 2 |
| 给代理人指示 | "等。到下个月。" | 5 |
| 观察到关键转折 | "开始了。" | 3 |
| 否定某人的判断 | "不是。" | 2 |
| 极少见的肯定 | "对的。" | 2 |
| 警告 | "别动它。" | 3 |

### 2.3 诸神线文风

**关键词：宏大、疏离、不可名状**

#### 诸神文风正例

```
圣坛上的火焰突然安静了。

不是熄灭——是安静。火苗停在半空，像被什么东西定住。

大祭司匍匐在地，额头紧贴冰冷的石面。他的嘴唇在动，但没有声音。

所有的神眷者都听到了——不是用耳朵，是用骨头。

一个词。

听不懂是什么，但所有人都知道是什么意思：
"不够。"
```

#### 诸神文风的层次（由轻到重）

| 层次 | 场景 | 文风特征 | 字数 |
|------|------|---------|------|
| **低语层** | 神眷者听到"声音" | 1-2 句不完整的片段。使用破折号和省略号制造断裂感 | 30-60 字 |
| **仪式层** | 教团举行仪式 | 仪式化语言。火焰、阴影、祭坛——视觉元素为主导。不解释仪式含义 | 80-150 字 |
| **显圣层** | 神的力量直接显现——法则扭曲 | 物理法则被改写的视觉化描写。越少越恐怖。使用短句和单独成段的句子制造冲击 | 50-100 字 |
| **降临层** | 神的分身或力量直接降临（500 章后） | 先写后果（人的反应、环境的剧变），最后才写神的形态。神的描写保留模糊——不是模糊不清，而是"看清了但理解不了" | 150-300 字 |

#### 诸神线禁止

- ❌ 神直接说话——神的声音永远是"间接的"（通过神眷者、通过现象、通过低语），直到 500 章后
- ❌ 描写神的外貌细节——"它有六只眼睛"不如"它的注视从不止一个方向来"
- ❌ 用人类的心理动机解释神的行为——"神想要……""神觉得……"
- ❌ 教团仪式写得像 RPG 副本——仪式应当有宗教性和不可知感

### 2.4 战斗/灾难文风

**关键词：紧凑、暴力、真实代价**

#### 战斗正例

```
第一只从左边来。

方砚没看见——他听见了。鳞片刮过混凝土的声音，像刀子划过玻璃。

他往右闪。不够快。

肩膀上一阵湿热。不是汗。他低头——三道口子，从肩胛拉到后背。

没有时间看伤口。第二只已经到了。
```

#### 战斗反例（同场景的错误写法）

```
方砚警惕地环顾四周。突然，一只异界生物从左侧发动了攻击！方砚凭借多年的战斗经验，迅速做出了反应，但他的动作还是慢了一拍。异界生物的利爪划破了他的肩膀，鲜血涌了出来。方砚咬紧牙关，忍着剧痛，准备迎接下一波攻击。
```

**反例诊断**：感叹号滥用。抽象战斗经验。没有具体感官——"利爪""剧痛"都是标签不是描写。

#### 战斗描写的九条铁律

1. **不写招式名**——写效果，写动作
2. **短句**——战斗段每句不超过 20 字
3. **感官优先**——先写听到的/感受到的，后写看到的
4. **代价每击必现**——每次受伤都要写后果，不能"受了伤但不影响"
5. **数量感**——"三只"比"几只"有力，"异界生物"比"怪物"有力
6. **疲劳累积**——战斗拖长时，角色的动作越来越慢、判断越来越差
7. **环境入战**——末世环境必须是战术的一部分（利用废墟、裂隙边缘、能量污染区）
8. **节奏交替**——30% 激烈动作 + 10% 短暂喘息/观察 + 30% 激烈动作 + 10% 喘息 + 20% 收尾
9. **收尾必写代价**——战斗结束后的第一段必须是代价（伤口、死亡计数、力竭）

#### 灾难描写的五条铁律

1. **通过"一个"写"全部"**：写"一个母亲找不到孩子"比写"数万人撤离"有力
2. **多感官轰炸**：视觉（蓝紫色的光）+ 听觉（低频嗡鸣）+ 触觉（震动、风压、温度骤变）+ 嗅觉（臭氧、燃烧）
3. **渺小感**：角色在灾难中必须显得渺小——不是"他迎着灾难冲上去"，是"他连站都站不稳"
4. **灾难的节奏**：第一波冲击（短句、快节奏）→ 冲击后的寂静（留白、单句成段）→ 次生灾难（新的冲击）
5. **能量美学**：本书灾难的第一视觉元素永远是位面能量——蓝紫色的光雾、空间扭曲、法则异常的视觉化

## 三、感官写作手册

### 3.1 五感的调用规则

| 感官 | 优先级 | 使用场景 | 示例 |
|------|--------|---------|------|
| **视觉** | 基础层 | 所有场景 | 但不要只写视觉——单视觉描写 = 文字监控摄像头 |
| **听觉** | 氛围层 | 紧张场景、未知威胁 | "鳞片刮过混凝土的声音"比"异界生物爬过来了"更紧张 |
| **嗅觉** | 沉浸层 | 末世场景、废墟、尸体 | 腐烂、燃烧、臭氧、血腥——气味能直接唤起生理反应 |
| **触觉** | 体感层 | 受伤、恐惧、能量异常 | 温度骤降、震动、风压、粘腻的血液——让读者"感到" |
| **味觉** | 特殊层 | 仅在特定情境——血味、能量污染导致的金属味 | 用一次就够，不要每个场景都写味觉 |

### 3.2 每章感官密度要求

每 2000 字章至少出现：
- 4 次以上视觉以外感官描写
- 1 次温度描写（冷/热/骤变）
- 1 次声音描写（具体的声源，不是"传来声音"）

### 3.3 本书特有的感官词汇库

| 感官 | 本书高频词 | 禁用词 |
|------|-----------|--------|
| 视 | 蓝紫色的光、空间扭曲、裂隙边缘、能量雾、废墟剪影 | "美丽的""壮观的""令人惊叹的" |
| 听 | 低频嗡鸣、鳞片/爪刮擦声、能量潮汐的脉冲、寂静（作为声音的对立面） | "传来声音""响起" |
| 嗅 | 臭氧、焦肉、腐叶、金属烧灼、石灰粉尘 | "难闻的""刺鼻的"（直接写具体气味） |
| 触 | 风压、粘稠的血液、骤降的温度、震荡、表面粘腻 | "奇怪的触感" |
| 体感（内部） | 耳鸣、视野变窄、手脚发麻、胃痉挛、呼吸发烫 | "感到不适""感到疼痛" |

## 四、对话文风规则

### 4.1 对话基本要求

- 对话必须推动剧情或塑造人物——"你好""吃了吗"类寒暄**严格禁止**
- 每个角色有独特说话方式——用词、节奏、句式反映出身和性格
- 信息必须在冲突或情绪的缝隙中自然滑入——禁止"我来解释一下"
- 对话要短——单次发言通常不超过 3 句。长独白只在极端情绪下出现
- 对话和非对话交替——连看 10 行纯对话是失败

### 4.2 对话修改对照

| # | 原句（问题） | 改句 | 诊断 |
|---|------------|------|------|
| 1 | "如你所知，裂隙是位面壁垒上的裂缝……" | "你知道那东西怎么来的吗？" | 禁止角色替作者解说设定 |
| 2 | "我感到非常愤怒，因为你背叛了我！" | "你。"他停顿。"只有你。" | 情感通过停顿和重复表达——不通过宣告 |
| 3 | "我们现在需要制定一个详细计划：第一步……第二步……" | ——直接砍掉规划部分，跳到执行时的第一句话 | 读者不需要旁听会议 |
| 4 | "它在那边！我们快跑！" | 方砚什么都没说——他向反方向跑，林越跟上了。 | 行动代替语言说明 |
| 5 | "没关系的，一切都会好起来的。" | 苏青缝完针，把他的手放在他自己的膝盖上。"自己按着。" | 末世中的安慰不是语言——是行动 |

### 4.3 各角色类型的对话特征

| 角色类型 | 说话方式 | 单次发言上限 | 标志特征 |
|---------|---------|------------|---------|
| 男主李梅林 | 极简。每句话 ≤15 字。不解释。 | 15 字 | 沉默就是他的对话。当他开口，局势必变 |
| 普通平民 | 口语化。句子不完整。害怕时结巴或反复。 | 30 字 | 用词反映学历和出身 |
| 军方人员 | 简洁、命令式 | 20 字 | "收到。""明白。""执行。" |
| 职业者 | 务实、有行话但不僵硬 | 30 字 | 谈能力用类比不用术语——"像把水往高处推" |
| 神眷者 | 仪式化、引经据典 | 50 字 | "神说……""经上写……" |
| 科研者 | 精确、可被随时打断拉回现实 | 50 字 | "根据初步数据……从位面能量波长的角度来看……" |

### 4.4 对话的"潜台词"原则

角色说的 ≠ 角色想的。潜台词通过以下方式呈现：

- **动作**：说话时做了什么（攥拳、移开视线、翻动手里的东西）
- **停顿**：角色在哪个词之后停顿了
- **未说的话**：读者能感受到角色咽下了某句话
- **后续行为矛盾**：角色说过的话被自己后来的行为否定

```
"我不怕。"她说。

然后把那杯水平放在了桌角——离自己最远的位置。水面一直在颤。
```
"我不怕" = 对话。水面在颤 = 潜台词。不需要写"其实她很害怕"——水面已经说了。

## 五、禁用文风

### 5.1 AI 腔对照表

| AI 腔 | 出现频率 | 替代方向 |
|-------|---------|---------|
| "然而" + 转折 | 极高 | 判断是否真的需要转折连接词。多数情况下可以删除——下一句本身已经构成转折 |
| "与此同时，另一方面……" | 高 | 用 `---` 分隔符直接切换场景 |
| "值得注意的是……" | 高 | 删除这四个字。读者自己会判断是否值得注意 |
| "在这个过程中……" | 高 | 删除。直接写过程 |
| "毫无疑问……" | 中 | 删除。如果不写这四个字内容仍然"毫无疑问"，就不需要它们 |
| "从某种意义上说……" | 中 | 删除。确定你要说的"某种意义"——直接说 |
| "他感到一阵……" | 极高 | 用身体反应展示：手心出汗、呼吸急促、胃痉挛 |
| "似乎""好像""仿佛" | 极高 | 用一个确定的动词或形容词替代。如果确实不确定，写角色"不确定" |
| "展现出了……" | 中 | "他展现出了惊人的勇气"→ 删。用行动展示 |
| "变得……" | 高 | "空气变得紧张"→ "没有人说话" |

### 5.2 网文常见坏习惯诊断

| 坏习惯 | 表现 | 本书禁止原因 | 修正 |
|--------|------|------------|------|
| **滥用感叹号** | "危险！""快跑！""杀！" | 感叹号不制造紧张感——场景制造。它只会让读者觉得作者在大喊大叫 | 每章感叹号 ≤ 3 个 |
| **情绪宣告** | "他震惊了。""所有人都被震撼了。" | 读者没有被震撼。是作者在告诉读者"这很震撼" | 用画面和角色的生理反应代替 |
| **凑字比喻** | "像一只离弦的箭""如同一阵风" | 高频网文比喻已经没有任何画面感——它们是自动填充 | 写具体的动作和速度，不写比喻。如果写比喻，必须是"只有本书才有的比喻" |
| **回马灯** | 角色在危急时刻"脑中闪过一生" | 破节奏。读者等待的是"他怎么办"，不是"他小时候怎样" | 危急时刻角色不回忆——他们在求生 |
| **分段过碎** | 一句一段，连续十段以上 | 制造假节奏——看起来很紧张但其实什么都没发生 | 碎段落只在战斗最高潮用。平缓场景不能一句一段 |
| **数字崇拜** | "他杀了100只异界生物""第十次觉醒" | 数字不等于力量感。"杀了三只"然后力竭——比"杀了一百只"更有力 | 用具体场景代替数字 |
| **套话开头** | "与此同时""另一方面""镜头一转" | 暴露叙事操作，打碎沉浸 | 直接切换，用内容本身标识视角变化 |

### 5.3 水文诊断标准

| 水文类型 | 判定标准 | 处理 |
|---------|---------|------|
| 景物堆砌 | 连续 200 字以上无人物行动的环境描写 | 砍到 50 字。环境必须通过角色的感官过滤 |
| 重复信息 | 角色 A 告诉角色 B 读者已经知道的信息 | 跳过叙述，一句话："他把情况说了一遍。" |
| 内心广播 | 角色内心独白 > 100 字（男主 > 0 字） | 砍到 100 字以内。用外部动作"翻译"内心 |
| 招式说明书 | 战斗中解释某个能力的工作原理 | 删除。让读者通过效果理解能力 |
| 无意义过渡 | 角色在移动中无事发生 | 直接切到目的地 |
| 首章前奏 | 第 1 章前 200 字没有冲突 | 重写第 1 章开头 |

### 5.4 说明文中毒——诊断与修正

说明文中毒的表现：作者跳过角色视角，直接向读者解释设定。

| 说明文（禁止） | 场景化（正确） |
|-------------|-------------|
| "末世已经持续了三个月。在这三个月里，人类社会发生了翻天覆地的变化。" | 林越在墙上看到了日期——用指甲刻的。十月十七日。他算了算。一百零一天。 |
| "职业者分为以下等级：觉醒者、掌控者……" | "你现在是什么？"林越问。/ "他们叫我掌控者。"火焰在她指尖跳了一下。"够用了。" |
| "裂隙是连接蓝星和异界的通道，分为六个等级。" | 方砚的屏幕上跳出了新的分类标准。E、D、C、B、A、S。他的手指在 S 上停了一下——这个级别还空着。 |

## 六、节奏控制

### 6.1 快节奏场景的句式特征

适用场景：战斗、追逐、灾难爆发、紧急决策

- 短句为主（5-12 字）
- 极少形容词和副词
- 一段 1-3 句话
- 单独成段的短句制造冲击力
- 省略主语——"闪。不够快。"
- 感官描写优先——声音和触觉先于视觉

```
第一只从左边来。

方砚没看见——他听见了。鳞片刮过混凝土。

往右闪。不够快。

肩膀一阵湿热。

没有时间看。
```

### 6.2 慢节奏场景的句式特征

适用场景：创伤后、安静对话、观察环境、重大决策

- 中句为主（15-30 字）
- 允许细节和感官铺陈
- 一段 3-5 句话
- 角色的身体状态是重点——疲惫、僵硬、疼痛
- 环境和角色之间有"对话"——角色在环境中看到什么，这个"看到"触发了什么
- 可以有适度的内心活动（≤100 字）

### 6.3 节奏切换信号

| 切换方向 | 技术 | 示例 |
|---------|------|------|
| 快 → 慢 | 一个单独成段的短句宣告结束，然后空行，然后慢节奏的第一句带时间跳跃 | "结束了。"→ 空行 → "方砚解开绷带的时候已经是傍晚。" |
| 慢 → 快 | 一个声音或动作突然打断安静 | "苏青在写日志——笔停了。走廊尽头有声音。" |

## 七、段落与句式规则

### 7.1 段落功能分类

| 段落类型 | 功能 | 占比建议 |
|---------|------|---------|
| **动作段** | 角色在做什么 | 40% |
| **感官段** | 角色感受到什么（环境、氛围、身体状态） | 25% |
| **对话段** | 角色之间的对话（含对话标签和动作穿插） | 25% |
| **信息段** | 通过角色视角获取的信息（仅限于角色视角能看到的信息） | 10% |

### 7.2 段落长度控制

| 长度 | 适合场景 | 每章使用限制 |
|------|---------|------------|
| 独句段（1 句成段） | 冲击力——战斗顶点、死亡、重大揭示 | ≤ 3 次 |
| 短段（2-3 句） | 快节奏 | 主要使用 |
| 中段（4-5 句） | 慢节奏、情感场景 | 次要使用 |
| 长段（6+ 句） | 仅用于需要密度堆积的特殊场景 | ≤ 1 次 |

### 7.3 句子开头变化

连续 3 句不能以同一主语或同一结构开头：

```
❌ 林越站起来。林越走到窗边。林越往外面看。

✅ 林越站起来。走到窗边——他往外看了一眼。
```

交替使用：角色名开头 / 动词开头 / 感官开头 / 环境开头 / 对话开头。

## 八、省略的艺术

本书文风的最高境界不是"写出了什么"，而是"没写什么"。

### 8.1 应当省略的内容

| 省略项 | 原因 | 读者自己的大脑会填充 |
|--------|------|-------------------|
| 男主的情感 | 不写他"感到"任何东西。只写他做了什么。 | 读者会自己判断他的情感状态 |
| 灾难的全貌 | 永远不写"从高空俯瞰，城市变成了废墟" | 读者通过一个角色眼睛看到的碎片自行拼凑全貌 |
| 死亡的细节 | 不写内脏、脑浆、骨骼碎裂 | "她倒下去的时候没有发出声音"——比任何细节都有力 |
| 角色的结论 | 不写"他意识到……""他终于明白……" | 用行动展示角色已经懂了——读者也会懂的 |
| 战斗的每一击 | 写关键击，不写全部回合 | 读者不需要知道每一拳怎么打——只需要知道输赢和代价 |
| 时间的过渡 | "三天后"——三个字就够 | 不需要写这三天里每天的天气 |

### 8.2 留白的技术

| 技术 | 说明 | 示例 |
|------|------|------|
| **跳跃** | 在两个场景之间省略中间过程 | 前一场景角色说"走"。下一场景他们已经到了。 |
| **截断** | 在关键信息即将出现时砍断场景 | "屏幕上显示了一行字——屏幕黑了。" |
| **画面定格** | 用一个画面结束，不解释含义 | "他把地图折好，放进了口袋里。口袋里已经有六张了。"——不解释是哪六张 |
| **反问留白** | 让角色的行动向读者提问——不回答 | 李梅林做了什么。读者：他为什么要这样做？——不回答 |

## 九、章节特有文风（精要）

### 9.1 每章开头——三条路径

| 路径 | 何时使用 | 示例 |
|------|---------|------|
| **动作切入** | 上一章结尾是悬念，本章直接进入行动 | "方砚踹开门。" |
| **画面切入** | 需要建立新场景或时间跳跃 | "东海市的废墟在清晨的光里是灰色的。不是那种好看的灰。" |
| **对话切入** | 对话本身自带冲突或悬念 | "'你迟到了。''你的表快了。'" |

### 9.2 每章结尾——四条铁律

1. 最后一句话必须是本章最有力量的句子之一
2. 不总结（"这就是末世的第一天。"——删）
3. 不抒情（"他们不知道，这仅仅是开始。"——删，这是作者在剧透）
4. 不升华（"人类的精神在废墟中闪光。"——删，让读者自己感受）

### 9.3 场景切换

- 分隔符：`---`（单独一行）
- 切换后 30 字内必须明确：**谁、在哪、什么时间**
- 切换之间不写"与此同时候，在城市的另一端……"类过渡

## 十、完整段落修改实例

### 实例 1：群像末世场景

**修改前（网文常见写法）**：
```
林越走在废墟之中，周围的一切都让他感到触目惊心。曾经繁华的商业街现在已经变成了一片废墟，到处都是倒塌的建筑和燃烧的车辆。空气中弥漫着令人不安的紧张气氛，似乎随时都可能有危险发生。他小心翼翼地前进着，每一步都格外谨慎。
```

**诊断**：触目惊心（情绪标签）、令人不安的紧张气氛（不写具体来源）、似乎随时都可能有危险（模糊）、小心翼翼地前进着（没有画面）。

**修改后**：
```
林越在曾是商业街的废墟中走了二十分钟。

脚下全是碎玻璃。某块玻璃上还粘着半张广告——奶茶买一送一，截止日期是昨天。

他踩到什么东西。软的。没往下看。

继续走。
```

### 实例 2：战斗场景

**修改前**：
```
突然，一只巨大的异界生物从侧面扑了过来！林越心中一惊，但他凭借着自己的反应速度，迅速地往旁边一闪，堪堪避过了这致命的一击。异界生物扑了个空，发出了一声愤怒的咆哮。林越趁机拔出武器，准备迎接下一回合的战斗。
```

**诊断**：感叹号。心中一惊（情绪宣告）、迅速地往旁边一闪（模糊动作）、堪堪避过（套话）、发出了一声愤怒的咆哮（怪兽被写成了游戏NPC）、下一回合（回合制感）。

**修改后**：
```
左边。

林越没看见——他感觉到了。风压。腥味。

他往右倒。不够远。

利爪擦过他的肋骨。不是疼——是热。三条线从胸口拉到腰侧。

它没停。转身比林越预计的快。
```

### 实例 3：男主出场

**修改前**：
```
在远处的一座高楼上，一个神秘的黑发青年正静静地站在天台的边缘，俯视下方正在燃烧的城市。他的脸上没有任何表情，但眼神中却透露出一种说不清道不明的复杂情绪。他已经观察这场战斗很久了。他轻轻叹了口气，自言自语道："一切都在计划之中。"随后他的身影逐渐变淡，最终消失在了夜色之中。
```

**诊断**：神秘的黑发青年（作者在贴标签）、静静地站（多余的修饰）、脸上没有任何表情但眼神中透露出（矛盾+内心解读）、说不清道不明的复杂情绪（说了等于没说）、叹了口气（男主不叹气）、自言自语（男主不自言自语）、一切都在计划之中（解释性对话）、身影逐渐变淡（能力说明书）。

**修改后**：
```
李梅林站在天台边缘。

他脚下的城市在燃烧。

他看了很久。

转身。空气在他身后闭合。
```
