# 14 — 番茄小说平台连载策略

> 本文档为《是我带来了末世》在番茄小说平台发布的全周期策略。不是营销指南——而是写作时的平台适配意识。

---

## 一、平台核心规则

### 1.1 番茄小说特点
- **免费阅读 + 广告变现**：读者免费看，平台通过广告赚钱
- **读完率至上**：读完率（完读率）是番茄最核心的推荐指标
- **前 10 万字定生死**：如果前 10 万字读完率低，后面基本没有推荐
- **章节单价**：广告分成与阅读时长挂钩——章节太短或读完率低会直接影响收入
- **推荐机制**：算法推荐，新书有"新书期"流量扶持

### 1.2 对写作的核心影响
1. 每章必须有"翻下去"的动力——读完率是生命线
2. 前 30 章是生死线——必须全力保住读者
3. 每章不宜太短——推荐算法倾向于 2000-2500 字/章
4. 标题决定了读者是否点进来——"标题党"不是贬义，是生存技能
5. 开头 200 字决定了读者是否继续看——第一印象只有一次

---

## 二、章节节奏策略

### 2.1 每章结构

```
【钩子开头】—— 0-50 字，冲突/悬念/反常/情感
【剧情推进】—— 1500-1800 字，核心事件
【爽点/燃点】—— 插入在剧情推进中
【钩子结尾】—— 最后 50 字，翻章动力
```

### 2.2 章节节奏密度

| 位置 | 节奏要求 |
|------|---------|
| 第 1 章 | 200 字内建立世界观核心和冲突。必须有"这是什么"和"为什么我要看"的答案 |
| 前 5 章 | 每章都有新信息、新悬念。必须有至少 1 个"让你想追"的角色 |
| 前 10 章 | 必须出现末世的"残酷面"（一个死亡、一次失败）。不能全是主角光环 |
| 前 30 章 | 必须有 3 个以上"小高潮"和 1 个"大高潮"。让读者有"这本书好看"的确信 |
| 前 100 章 | 不能出现连续 2 章以上的"没有实质推进" |

### 2.3 章末钩子类型

| 钩子类型 | 示例 | 使用频率 |
|---------|------|---------|
| 危险钩子 | "门被推开了。" | 高频 |
| 悬念钩子 | "那个名字——他认识。" | 高频 |
| 反转钩子 | "躺在地上的不是他。" | 中频 |
| 情感钩子 | "'他还活着。'她说完这句话，哭了。" | 低频（不要滥情） |
| 信息钩子 | "屏幕上的数字不对。" | 中频 |
| 人物钩子 | "她摘下面具。" | 中频（角色登场/身份揭示） |

## 三、前 30 章留存策略（关键）

### 3.1 第 1-3 章：砸世界观
- 第一句话开始就是末世——不铺垫
- 用画面和事件展示世界观，不解释
- 在 200 字内让读者知道：这不是游戏末世，不是丧尸末世
- 建立第一个"想追"的角色

### 3.2 第 4-10 章：铺悬念 + 展示残酷
- 展示末世真实的代价——有人死、有人疯、有人绝望
- 抛出核心悬念——"末世到底从哪来的？"
- 第一次暗示"这一切不自然"
- 男主首次"被读者看到"（角色看不到）

### 3.3 第 11-20 章：群像展开 + 力量体系初显
- 第二条 POV 线展开
- 职业者正式亮相
- 第一次"爽点小高潮"——一场成功的战斗
- 抛出"职业者 vs 普通人"的初期矛盾

### 3.4 第 21-30 章：强化差异化 + 首次大高潮
- 进一步暗示"末世有人为痕迹"
- 第一次大型战斗（多方视角）
- 第一次"男主出手的间接后果"
- 第一卷第一个卷内爆点

## 四、简介与标题策略

### 4.1 简介方向（供写作完成后提炼）

简介应该突出本书的差异化——**"末世是男主带来的，但这是他拯救蓝星的唯一方式"**。

推荐简介类型：
- **设问型**："如果末世不是天灾，是人为——你会感谢他还是恨他？"
- **反转型**："所有人都以为末世的到来是末日。只有他知道——这才是蓝星唯一的生机。"
- **人设型**："一个亲手打开末世的穿越者。一个全人类都不知道的秘密。一场注定被误解的拯救。"

### 4.2 单章标题风格

番茄小说平台适合的标题风格：
- 信息量 + 冲突感 + 适度神秘
- 不要太文艺（"春日里的思念"在番茄会死）
- 不要太直白但也不要太含蓄
- 可以从本章最"高能"的片段中提炼

标题示例：
- "第 1 章：天空裂开了"
- "第 5 章：它不在这个世界"
- "第 8 章：他在天台上看着"
- "第 15 章：火从掌心里涌出来"
- "第 30 章：它比昨天更大了"

### 4.3 关键词避雷
番茄小说有内容审核机制。避免以下敏感内容：
- 过于具体的现代政治影射
- 过于血腥的细节描写（可以写死亡，但不能写分尸细节）
- 色情内容
- 宣扬反社会价值观

## 五、读完率意识

### 5.1 每章读完率的敌人

| 敌人 | 表现 | 对策 |
|------|------|------|
| 开头无聊 | 环境描写堆砌、说明文开篇 | 直接冲突或悬念 |
| 中段注水 | 无信息增量的过渡段落 | 砍掉或合并 |
| 结尾无力 | 没有钩子、自然结束 | 必须留钩子 |
| 信息倾倒 | 大段设定说明 | 融入场景 |
| AI 腔 | "然而""与此同时""值得注意的是" | 严格禁止 |
| 角色无趣 | 没有性格、没有动机、没有缺点 | 每个角色都有真实欲望 |

### 5.2 读完率的盟友

| 盟友 | 表现 |
|------|------|
| 好奇心 | "到底怎么回事？"——让读者不停追问 |
| 投入感 | 群像角色的命运被揪住 |
| 爽感 | 每章的突破/反转/战斗胜利 |
| 期待感 | 对下一章的确定性期待 |
| 共鸣感 | 角色的恐惧、希望、牺牲——感同身受 |

---

## 六、风险规避

### 6.1 同质化风险
市面上存在大量末世/系统/转职类作品。本书最大的差异化在于：
- **男主就是末世制造者**——这不是末世求生，这是末世真相
- **末世是锻炼手段**——不是打怪升级，是文明备战
- **终极敌人是神**——维度比一般末世高

在写作中，每一次都应该强化这个差异化。不要在早期过多描写"异界生物什么样"，而要更多描写"这一切为什么不自然"。

### 6.2 敏感内容规避
- 不明确写中国/美国/任何现实国家的具体决策
- 使用虚构城市名（如"东海市"）
- 学术机构、军方、政府的行为写实但不涉及现实敏感信息

### 6.3 读者疲劳规避
- 连续 3 章以上同一个角色的 POV → 读者会疲劳
- 连续 3 章高强度战斗 → 读者会麻木
- 连续 3 章没有信息增量 → 读者会弃书
- 连续 3 章没有男主暗示 → 核心读者会失去"等待男主"的乐趣
