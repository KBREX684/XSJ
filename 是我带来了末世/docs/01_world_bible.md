## 一、世界真相

### 1.1 多位面宇宙结构

宇宙由无数**位面**构成，位面之间存在等级差异：

| 层级 | 名称 | 特征 | 代表位面 |
|------|------|------|---------|
| 神级 | 神域 | 诸神居所，规则由神明书写 | 诸神领域 |
| 高级 | 高等位面 | 魔法文明高度发达，职业体系成熟 | 李梅林原世界（已毁灭） |
| 中级 | 本位面 | 科技文明为主，无魔法，无职业体系 | 蓝星 |
| 低级 | 低等位面 | 蛮荒、异化、能量紊乱 | 裂隙中的异界 |

### 1.2 位面通道

- 位面之间的通道在正常情况下是封闭的
- 高等存在的注视可以"软化"位面壁垒
- 男主李梅林用原世界的秘术，主动在蓝星和多个低等位面之间打开了通道
- 通道的开启导致低等位面的能量和生物涌入蓝星——这就是"末世"的物理本质
- 通道也带来了原世界职业体系的**法则种子**——使蓝星人类得以觉醒职业力量

### 1.3 诸神与位面收割

- 诸神并非创造者，而是**位面收割者**
- 当一个位面的文明发展到足以被注意到时，诸神会评估其"价值"
- 有价值的位面会被吸收、改造、奴役
- 没有价值的位面会被"清理"——即毁灭
- 李梅林的原世界是一个"有价值"的位面，被诸神征服后最终毁灭
- 诸神现在正在评估蓝星——评估期约为蓝星时间 10 年
- 一旦评估完成，结果只有两个：被征服或被毁灭

## 一-A、奥拉诺斯：已毁灭的高等位面

### 1A.1 文明概况

奥拉诺斯是梅林（在蓝星名为李梅林）的原世界。在被诸神毁灭之前，它是一个存续了约十二万年的高等魔法文明。

| 维度 | 特征 |
|------|------|
| 文明类型 | 魔导文明——将魔法与工程学一体化的社会形态 |
| 技术水平 | 超出蓝星当前水平数个层级。能源来自位面能量，建筑与自然融合，交通依赖空间折叠技术 |
| 政治形态 | 联邦议会制，七个主要城邦联合治理。没有国王，没有皇帝，由大魔导师组成的"环议会"决策 |
| 军事体系 | 以职业者军团为核心。在神战爆发前，奥拉诺斯常备七支军团，每支以一位法则级以上的职业者为军团长 |
| 人口 | 鼎盛期约五亿 |
| 与诸神的关系 | 毫不知情——直到评估期结束时才发现有"更高的存在"注视着他们 |

### 1A.2 职业体系的发源

奥拉诺斯是人类已知第一个也是唯一一个系统化了"职业法则"的文明。经过数万年的研究和实践，奥拉诺斯的学者发现：

- 位面能量可以被智慧生命的灵魂"锚定"
- 不同灵魂天然与不同的法则碎片契合
- 通过特定的训练路径，灵魂可以从"借用法则"进化到"理解法则"，最终"对话法则"
- 职业等级的本质是灵魂与法则的融合深度

这套体系不是奥拉诺斯人的"发明"——它是对宇宙基本法则的"发现"和"工程化应用"。梅林将其称为"法则工程学"。

### 1A.3 奥拉诺斯神战

诸神对奥拉诺斯的评估期约为数百年。在评估期间，诸神做了与现在对蓝星做的相同的事——通过神眷者和教团悄悄渗透。

评估结束后，诸神判定奥拉诺斯为"高级资源型位面"——这意味着值得征服和奴役，而非直接毁灭。

**神战过程**：

1. **渗透期（约200年）**：教团在奥拉诺斯边缘城邦活动，传播"神启"
2. **分裂期（约100年）**：几个城邦被教团实际控制，环议会分裂
3. **降临期（约50年）**：第一位神直接降临——"战争之神"科尔塔。降临的代价是所在区域被法则扭曲为不适宜任何生命存在的死域，数百万人瞬间死亡
4. **全面战争期（约80年）**：职业者军团与降临的神正面对抗。初期职业者取得局部胜利——在法则级职业者的配合下，击杀了一位神的"分身"。但真正的神不受物理伤害——他们需要被"法则级"的力量才能真正威胁到
5. **崩溃期（约20年）**：多位神同时降临。环议会崩溃。职业者军团被各个击破
6. **终末**：奥拉诺斯位面被彻底"清理"——神的法则覆盖了奥拉诺斯的原生法则。位面的物理规则被重写。一切生命终结

梅林是最后一位仍在战斗的大魔导师。他在环议会最后一座塔中亲眼看着奥拉诺斯的天空被法则撕裂。他是在奥拉诺斯法则被完全覆盖之前，强行通过位面壁垒逃离的唯一幸存者。

### 1A.4 奥拉诺斯留给蓝星的遗产

梅林带走的不是物品——是**知识**。他脑中保存着：

- 完整的三万七千种职业的觉醒路径和进阶图谱
- 法则工程学的全部理论
- 神战中收集的对诸神法则的分析数据
- 一个被证实的结论：**职业者文明理论上可以对抗神，但奥拉诺斯败在时间不足、准备不足、内部被渗透**

奥拉诺斯文明留给蓝星的不是一个可以复制的旧世界，而是一个关于"如何不重蹈覆辙"的教训。

### 1A.5 梅林的幸存

梅林能够逃出奥拉诺斯，是因为他完成了奥拉诺斯历史上唯一一次被正式记录的**跨位面传送**。这不是空间系的常规传送——空间传送必须在同一位面内。跨位面传送需要同时操作空间法则和时间法则的碎片。

代价是：梅林的身体被位面壁垒严重损伤。他在抵达蓝星后花了近两年时间恢复——这段时间诸神已经开始评估蓝星。他赶上了。

## 二、蓝星末世的本质

### 2.1 末世不是天灾，是人为

末世不是陨石、不是病毒、不是核战争。末世是**位面通道被人为打开后产生的位面能量污染**。

位面能量涌入蓝星，在物理层面表现为：
- 部分区域空间不稳定，出现**裂隙**——通往低等位面的裂缝
- 低等位面生物通过裂隙进入蓝星——不是丧尸，而是异界生物
- 位面能量改变蓝星的物理法则——某些区域出现异常重力、异常温度、能量潮汐
- 位面能量激活了部分人类的**职业法则**——职业者由此诞生

### 2.2 末世的时间线特征

- **初期（第 1 卷）**：裂隙零星出现，异界生物涌入，社会秩序崩溃，职业者首次出现
- **扩张期（第 2-4 卷）**：裂隙增多扩大，能量污染加剧，幸存者建立新秩序，职业者体系逐步成型
- **战争期（第 5-8 卷）**：异界文明从裂隙中发动有组织的入侵，蓝星各方势力联合对抗，诸神教团公开化
- **终局期（第 9-10 卷）**：男主与诸神的全面对抗，蓝星文明的最终命运

### 2.3 末世中的死亡

- 死亡是永久的。没有复活，没有重生
- 异界生物杀死的人不会变成丧尸
- 部分位面能量富集区域可能出现**能量变异体**——被能量扭曲的人类或生物，但这不是丧尸，而是位面污染的产物

### 2.4 位面能量的物理表现谱系

位面能量涌入蓝星后不只有一种形态。不同能量形态对应不同的末世现象：

| 能量形态 | 表现 | 对蓝星的影响 |
|---------|------|------------|
| **游离态能量** | 肉眼不可见，弥漫于空气中 | 使人类的职业法则开始被激活。浓度越高，觉醒概率越大 |
| **凝聚态能量** | 可见的蓝紫色光雾或光斑 | 在裂隙附近聚集。直接接触有毒性——会导致细胞层面"能量灼伤" |
| **结晶态能量** | 固态能量晶体——"能晶" | 可以被收集和使用。是末世中最重要的资源之一，作为能源和职业者进阶材料 |
| **潮汐态能量** | 周期性爆发的能量脉冲 | 会导致电子设备瘫痪、人体短暂失能、异界生物狂暴化 |
| **法则态能量** | 直接表现为物理法则的扭曲 | 在大型裂隙附近出现——重力异常、温度反转、空间折叠 |

### 2.5 末世现象链

末世不是"一夜之间世界变了"。以下是蓝星从正常到末世全貌的渐变链：

| 阶段 | 时间 | 现象 |
|------|------|------|
| **裂隙涌现** | 第1天 | 全球数百个城市同时出现裂隙。大小不一，从针孔大到横跨街区的裂隙都有 |
| **首波冲击** | 第1-3天 | 异界生物涌入，大量平民死亡。军队反应不及。通讯瘫痪 |
| **能量扩散** | 第3-30天 | 游离态能量蔓延全球。第一批职业者开始觉醒。电子设备随机失效 |
| **法则畸变** | 第30-90天 | 大型裂隙周边出现物理法则异常区。E/D级裂隙频繁开合 |
| **生态入侵** | 第90-180天 | 异界生物开始适应蓝星环境。部分异界物种在蓝星繁殖。生态系统开始被改造 |
| **文明侵蚀** | 第6月-2年 | 低等位面的智慧种族开始有组织地通过裂隙进入蓝星，建立前哨 |
| **全位面污染** | 第2年以后 | 蓝星的大气层、海洋、土壤中均已检测到位面能量。蓝星的物理法则正在被"稀释"

## 三、诸神体系

### 3.1 诸神的本质

- 诸神不是全知全能的——他们是极其强大的位面征服者
- 诸神的力量来源于对位面的**法则掌控**
- 每个神掌控一种或多种**法则**（如：战争、死亡、秩序、混沌、生命、时空等）
- 诸神之间存在竞争和博弈——他们并不团结
- 诸神关注蓝星的程度取决于蓝星在评估中的"价值等级"

### 3.2 诸神与蓝星的互动

- 诸神在评估期不会直接降临（需要大量能量，且可能引起其他神的警惕）
- 诸神通过**神眷者**在蓝星建立影响力
- 神眷者是诸神在蓝星的代理人，拥有神明赐予的部分力量
- 神眷者的力量来源与职业者不同——神眷者的力量是"借来的"，职业者的力量是"觉醒的"
- 教团是神眷者建立的组织，目的是扩大神明在蓝星的影响力，为将来神的降临做准备

### 3.3 诸神在故事中的呈现

- 前 200 章：诸神不直接出现，仅通过神眷者、教团仪式、不可解释的异象间接存在
- 200-400 章：神眷者开始获得更多力量，教团公开化，诸神的名字和领域开始被提及
- 400-600 章：个别神开始进行小规模直接干预（如：通过神眷者降下分身）
- 600-800 章：诸神全面介入，神战爆发

### 3.4 已知诸神（个体档案）

以下诸神是蓝星在故事发展中会遭遇的主要神明。每位神拥有一个或多个法则领域、一个神国、以及独特的行为模式。

#### 1. 科尔塔（Khortha）——战争之神 / 钢铁之神

| 维度 | 描述 |
|------|------|
| 法则领域 | 战争、武器、冲突、军事策略 |
| 神国 | 铁王座领域——由不断改变的金属结构构成的无限战场 |
| 性格 | 好战但不愚蠢。他是诸神中最先被派去征服新位面的执行者。他的行为方式直接而粗暴，但背后有精确的计算 |
| 对蓝星的兴趣 | 高。他评估蓝星为"有军事潜力的位面"。他渴望在蓝星上展开新的征服 |
| 神眷者特征 | 被赐予钢铁之力和战斗本能。神眷者在战斗中的感知力超出人类极限 |
| 弱点 | 对非武力形式的对抗缺乏敏锐度。在奥拉诺斯神战中曾被诱入陷阱 |

#### 2. 薇拉（Veyra）——秩序之神 / 法则之眼

| 维度 | 描述 |
|------|------|
| 法则领域 | 秩序、法律、契约、预言 |
| 神国 | 法典圣殿——由无限交错的法条和契约构成的图书馆迷宫 |
| 性格 | 冷峻、精确、耐心极强。她不会仓促行动。她的信徒相信她知道一切将要发生的事——这并非完全错误 |
| 对蓝星的兴趣 | 高。蓝星的现代社会体系——法律、政府、契约——让她着迷。她认为蓝星是"完美的秩序试验场" |
| 神眷者特征 | 被赐予"预知碎片"——能看到近期未来的片段，但这能力会逐渐扭曲他们的时间感 |
| 弱点 | 对混沌无法预测。在奥拉诺斯神战中，混沌之神萨恩的策略多次让她失算 |

#### 3. 萨恩（Saan）——混沌之神 / 裂变之神

| 维度 | 描述 |
|------|------|
| 法则领域 | 混沌、裂变、无序、变化 |
| 神国 | 永动裂谷——一切形态和结构都在不断变化、崩溃、重组的领域 |
| 性格 | 不可预测。他有时破坏，有时创造，有时什么都不做。诸神中他是最难被理解的一个 |
| 对蓝星的兴趣 | 中。他认为蓝星太"有序了"。但末世的混乱正好吸引了他的注意 |
| 神眷者特征 | 被赐予让物理法则局部失效的能力。他们身边的事物会随机变化 |
| 弱点 | 无法制定长期计划——他的本性让他不擅长坚持 |

#### 4. 玛尔（Maal）——死亡之神 / 终焉之主

| 维度 | 描述 |
|------|------|
| 法则领域 | 死亡、时间终结、灵魂收割 |
| 神国 | 灰色彼岸——被描述为"一切事物走到的最后一站" |
| 性格 | 沉默。在奥拉诺斯神战中，其他神说过话，他从未开口。他的目的不是统治，不是征服——是"收录" |
| 对蓝星的兴趣 | 最高。蓝星有七十亿灵魂——他从未面对过如此规模的"收获" |
| 神眷者特征 | 被赐予触碰死亡的能力。他们可以感知即将死去的人，可以让死亡加速或暂时延缓 |
| 弱点 | 对"生命"的反面——过于热切的活力和希望——有天然的排斥。他曾因此被生命法则的职业者牵制 |

#### 5. 瑟瑞昂（Therion）——生命之神 / 繁衍之主

| 维度 | 描述 |
|------|------|
| 法则领域 | 生命、进化、繁衍、生物改造 |
| 神国 | 无限花园——每一寸空间都长满不断变异和进化的生物 |
| 性格 | 好奇、实验性、带有一种残忍的"科学精神"。他不关心个体生命——他关心的是"生命"这个抽象概念 |
| 对蓝星的兴趣 | 高。蓝星的生物多样性引起了他极大的兴趣。他想要"改良"蓝星的生态系统 |
| 神眷者特征 | 被赐予操控生物组织的能力。可以治愈或扭曲生命体 |
| 弱点 | 他的"实验"有时创造的东西会反噬他自己 |

#### 6. 虚空（Void）——不是神的存在

在诸神体系中存在一个异类。虚空不是一个神——它是一种**反法则**。如果诸神代表"法则的存在"，虚空代表"法则的缺失"。

- 没有意识，没有目的，只有吞噬
- 李梅林怀疑虚空是位面收割的"最终产物"——当所有的位面都被收割完，诸神自己也将面对虚空
- 在故事中，虚空不是角色，而是一种终极威胁的暗示

### 3.5 神之间的博弈

诸神不是铁板一块。他们对蓝星的关注出于不同的动机，而这些动机之间存在冲突：

| 神A | 神B | 冲突本质 |
|-----|-----|---------|
| 科尔塔 | 薇拉 | 战争破坏秩序 vs 秩序约束战争 |
| 薇拉 | 萨恩 | 秩序与混沌的永恒对立 |
| 玛尔 | 瑟瑞昂 | 死亡与生命的对立法则 |
| 科尔塔 | 萨恩 | 有目的的战争 vs 无目的混乱 |
| 所有神 | 虚空 | 法则存在对法则缺失的根本恐惧 |

这种博弈对蓝星是优势——诸神不会同时全力介入。他们会互相牵制。这正是李梅林计划的根本前提：**利用神之间的博弈争取时间**。

### 3.6 神眷者的力量机制

神眷者与职业者虽然都能产生超常效应，但二者的力量来源在本质上是不同的：

| 维度 | 职业者 | 神眷者 |
|------|--------|--------|
| 力量来源 | 自身灵魂与位面法则的契合 | 神明的"赐予"——实际上是法则的暂时租借 |
| 代价 | 体能、精神力、法则反冲 | 自我意志的逐渐丧失，最终人格被神覆盖 |
| 进化方向 | 从借用法则到理解法则，最终对话法则 | 从接收神的意志到成为神的容器 |
| 终极形态 | 法则级：可以与法则平等对话 | 神临体：神的一部分意志降临在其身上 |
| 能否反抗神 | 可以——职业者的力量是自己挣来的 | 不能——力量来源本就是神，不存在反抗的基础 |

**神眷者的堕落弧线**（标准路径）：
1. **初选**：在极度绝望或渴望中被神"选中"——通常发生在生死关头
2. **赐福**：获得神赐予的力量，体验"神的恩典"——此时感觉极好
3. **依赖**：力量每一次使用都加深了对神的依赖。不使用力量时会感到"空虚"
4. **侵蚀**：开始听到神的声音——不是语言，而是直觉般的"启示"。这时的神眷者往往认为自己"更有智慧了"
5. **替代**：自我意志被神的部分意志覆盖。外貌、行为、思维模式都开始改变
6. **神临**：神的一部分直接降临在神眷者身上——此时原来的那个人格已经几乎不存在了

**叙事意义**：神眷者不是简单的"反派"。他们中有的人在初选时是真正的绝望之人——神确实给了他们力量。这让他们的悲剧变得复杂。

## 四、男主制造末世的逻辑

### 4.1 为什么必须制造末世？

1. **诸神已经在评估蓝星**：蓝星被注意到是不可逆的。什么都不做 = 等待毁灭
2. **蓝星文明不具备反抗能力**：科技文明在诸神面前不堪一击。需要一种新的力量体系
3. **原世界职业体系是已知唯一能对抗诸神的力量**：李梅林亲眼见证过职业者在神战中能做什么
4. **位面能量是觉醒职业的必要条件**：蓝星原本没有位面能量，不能自然觉醒职业。只有打开位面通道，让能量涌入，才能让职业法则在蓝星生根
5. **低等位面是天然的练兵场**：裂隙带来的异界生物为职业者提供了成长所需的实战经验

### 4.2 末世是"可控"的吗？

- 李梅林的目标是"可控末世"——尽量让末世在锻炼人类的同时不彻底毁灭文明
- 但实际上末世不会完全可控——李梅林自己也知道
- 大量平民死亡、城市毁灭、秩序崩溃——这些都是"代价"
- 李梅林接受这些代价。他没有被写成"痛苦的好人"——他做出了选择并承担后果
- 书中不回避"末世杀死了许多人"这一事实

### 4.3 男主的幕后操作方式

- 通过自己暗中建立的组织和代理人网络操纵末世进程
- 在关键节点出手确保文明不会彻底崩溃
- 将原世界的职业知识通过特定渠道传播给特定人物
- 监控诸神的动向，在神明干预升级前提前布局
- **他永远不会公开自己的身份和目的**——直到故事末期

## 五、位面裂隙

### 5.1 裂隙的物理特性

- 裂隙是位面壁垒上的裂缝，连接蓝星和某个低等位面
- 裂隙大小不一：小的仅容一人通过，大的可以让整座城市覆盖在位面能量之下
- 裂隙可能在任何地点出现，但高能量区域更不稳定
- 裂隙可以自然闭合，也可能被外力扩大或缩小
- 大型裂隙的周边区域会形成**能量污染区**——物理法则异常的地带

### 5.2 裂隙中的威胁

| 威胁类型 | 说明 |
|---------|------|
| 异界掠食生物 | 从低等位面涌入的捕食性生物，智力不高但危险 |
| 位面能量潮汐 | 周期性爆发的能量波动，可致人死亡或变异 |
| 能量污染区 | 裂隙周围法则扭曲的区域，可能改变重力、时间、物质状态 |
| 异界文明 | 有组织的低等位面智慧种族，后期会主动入侵蓝星 |
| 法则碎片 | 从高等位面掉落的法则残片，可能被职业者吸收强化 |

### 5.3 裂隙的等级

| 等级 | 描述 | 危险程度 |
|------|------|---------|
| E级 | 微型裂隙，容一人通过，几小时内自愈 | 低 |
| D级 | 小型裂隙，持续数天，偶有异界生物通过 | 中低 |
| C级 | 中型裂隙，持续数周，形成小型污染区 | 中 |
| B级 | 大型裂隙，持续数月，污染区范围广 | 高 |
| A级 | 巨型裂隙，可能永久存在，异界文明级威胁 | 极高 |
| S级 | 法则裂隙，连接含法则碎片的高等区域 | 灾难级 |

### 5.4 裂隙的动态行为

裂隙不是静态的洞。它们有自己的"生命"：

- **呼吸**：裂隙会周期性膨胀和收缩。膨胀期异界生物更容易通过
- **共振**：两个相近的裂隙之间可能发生共振——共振会同时扩大双方
- **愈合**：位面壁垒有自愈能力。E级和部分D级裂隙会在数小时到数天内自然闭合。越大的裂隙自愈越慢，A级以上可能永久存在
- **人为干预**：高等级职业者（大师级以上）可以用能力加速或减缓裂隙的愈合。空间系职业者可以强行关闭小型裂隙
- **锚定**：男主的秘术可以在特定位置"锚定"裂隙——防止它们自愈。这是确保末世持续的机制之一

### 5.5 异界生物谱系（非丧尸体系）

本书中不存在丧尸。以下是在裂隙中出现的异界生物类型：

| 类别 | 名称 | 特征 | 来源位面 | 威胁等级 |
|------|------|------|---------|---------|
| 掠食型 | 刃虫 | 昆虫形态，身体边缘锋利如刀。群居，数量优势 | 灰域 | E-D级威胁 |
| 掠食型 | 蚀影 | 不定形的暗影实体，在黑暗中几乎不可见 | 灰域 | D-C级威胁 |
| 巨兽型 | 裂蹄兽 | 体型如犀牛，但表皮由能量结晶覆盖，几乎免疫常规弹头 | 灰域 | C-B级威胁 |
| 寄生型 | 能量寄生体 | 不可见的存在，依附在活体上缓慢吸食生命力和位面能量 | 灰域 | D级威胁 |
| 智慧型 | 深岩族 | 人形生物，身披石质外壳。有社会结构，使用简单语言和工具 | 深岩位面 | B级威胁 |
| 智慧型 | 灰语者 | 灰域中的智慧个体。可以与人交流，动机不明 | 灰域 | C级威胁 |
| 法则型 | 法则碎片聚合体 | 由法则碎片构成的准生命体。对职业者有极强的吸引力和危险性 | 裂隙间 | A级威胁 |
| 神奴型 | 神造兵器 | 被某位神改造过的生物，专门用来测试新位面的反抗能力 | 未知 | S级威胁 |

### 5.6 裂隙的分布规则

裂隙不是完全随机的。它们的分布遵循以下规律：

1. **人口密度正相关**：裂隙更容易出现在人多的地方——因为神评估位面时，文明的聚集点是评估重点
2. **能量梯度效应**：一旦一个区域出现了裂隙，附近区域出现新裂隙的概率大幅上升
3. **地质无关**：裂隙与地质结构无关——它们可以出现在天空、地面、水下、地下
4. **子裂隙**：每个大型裂隙（C级以上）会周期性地在周边"释放"更小的子裂隙
5. **男主的设计**：李梅林在最初选择通道位置时，刻意避开了全球核武库、主要粮食产区和水源——这是他"可控"的部分

## 六、蓝星末世前后的变化

### 6.1 地理变化与关键区域

末世爆发后，蓝星的地理和社会图景被裂隙系统彻底改写。以下是故事中涉及的关键区域：

#### 6.1.1 东海市（第一卷核心舞台）

| 维度 | 设定 |
|------|------|
| 地位 | 虚构的沿海大型城市（参考上海/广州的规模），第一卷核心故事舞台 |
| 末世状况 | 遭受重创。市中心出现多个C级裂隙，沿海区域有B级裂隙 |
| 当前状态 | 城市西部被废弃。东部沿海为军方控制区。市中心为禁区 |
| 故事中的角色 | 林越的家乡。方砚所在部队的驻地。苏青工作的医院所在地 |

#### 6.1.2 蓝星全局状态

| 区域 | 末世状态 | 特征 |
|------|---------|------|
| 东亚沿海城市带 | 重灾区 | 人口密集+裂隙选择——双重灾难。东海市是其中之一 |
| 北美洲内陆 | 中等冲击 | 裂隙密度较低，但出现了数个大型荒野裂隙区 |
| 欧洲 | 分裂状态 | 各城市间裂隙阻断了传统交通线，各国各自为战 |
| 海洋 | 未知领地 | 水下的裂隙带来了完全不同于陆地的异界生物。船员报告称"海在变色" |
| 极地 | 异常安静 | 裂隙极少。原因不明。部分幸存者向北迁移 |

#### 6.1.3 安全区体系

末世后形成的聚居地分为以下几类：

| 安全区类型 | 特征 | 规模 | 例 |
|-----------|------|------|-----|
| 军方安全区 | 由军队控制，有围墙和防线。纪律严格 | 5万-50万人 | 东海市东区安全区 |
| 幸存者社区 | 由幸存者自发建立，组织松散，安全依赖职业者 | 500-5000人 | 郊区废校社区 |
| 公会据点 | 职业者公会的据点，专业度高，收费不低 | 1000-5000人 | 第一公会总部 |
| 自由镇 | 三不管地带，黑市活跃，法律模糊 | 不定 | "灰街" |
| 教团飞地 | 由神眷者教团控制的区域。外人出入受限 | 不定 | 第一卷末萌芽 |

### 6.2 社会变化

- 国家机构在初期遭受冲击后逐步调整。军方成为实质上的行政核心
- 传统警察系统在武器落后的情况下基本失效——职业者成为新的执法力量
- 职业者逐渐取代传统武装力量的核心地位。军队自身也在职业者化
- 经济体系崩溃。新货币是**能晶**——凝结的位面能量晶体，既可以用作能源，也可用于职业者修炼
- 科技文明并未消失——职业体系与科技开始融合。能晶电池、位面能量探测器、法则遮蔽装置等新技术出现
- 教育体系崩溃。新一代在末世中成长——对旧世界的记忆是"父母讲的故事"

### 6.3 通讯与物流

| 维度 | 末世前 | 末世后 |
|------|--------|--------|
| 全球通讯 | 互联网全覆盖 | 基本中断。局部有线网络尚存 |
| 区域通讯 | 手机网络 | 仅安全区内部有局部网络。长距离依赖短波无线电 |
| 长途运输 | 公路/铁路/航空/海运 | 基本停滞。裂隙阻断了大多数交通线 |
| 替代方案 | — | 空间系职业者的传送能力成为高端物流的核心 |
| 信息传播 | 媒体/社交网络 | 以"口耳相传"和无线电广播为主。谣言盛行 |

### 6.4 末世一年的关键时间节点

| 时间 | 状态 | 标志性事件 |
|------|------|-----------|
| 第1天 | 裂隙日 | 全球裂隙开启，末世爆发 |
| 第7天 | 第一周结束 | 大部分城市进入无政府状态。第一批职业者觉醒 |
| 第30天 | 第一个月 | 军方开始建立第一批安全区。能晶被首次发现 |
| 第90天 | 第一季度 | 职业者公会雏形出现。裂隙分级制被军方建立 |
| 第180天 | 半年 | 安全区体系基本稳定。教团开始在幸存者中活跃 |
| 第365天 | 第一年 | 第一次多势力联合清剿B级裂隙。第一个法则级别职业者出现 |

### 6.5 平凡人的末世生活

本书不回避平凡人在末世中的真实处境：

- **职业者与平凡人的力量差距**：觉醒者可以一人击杀异界生物，而平凡人连自保都做不到。这种差距产生了新的社会阶层
- **安全区内的生活**：有基本的水和食物配给。拥挤。每天有人死于疾病、饥饿、和"走出去碰运气"
- **安全区外的生存**：小团体求生。依赖对裂隙出现模式的了解和运气
- **对旧世界的怀念**：年长者。他们还能讲述"曾经的生活"——这在年轻人听来像神话
- **对末日的解释需求**：为什么这一切会发生？这是神的惩罚吗？是人为吗？——教团正是抓住了这个需求而兴起

## 六-A、位面评估：诸神的运行逻辑

### 6A.1 评估机制

诸神对位面的评估不是随意的——它遵循一套精确的运行逻辑：

| 阶段 | 持续时间 | 神的行动 | 位面的反应 |
|------|---------|---------|-----------|
| **发现** | 瞬间 | 神的目光"扫"到新位面 | 无 |
| **远程观测** | 数年-数十年 | 收集位面基础数据：文明等级、资源类型、人口规模、军事潜力 | 无——位面感知不到被"看" |
| **渗透** | 数十年-数百年 | 布设教团、选取神眷者、在内部制造分裂 | 社会出现不稳定，不正常的宗教兴起 |
| **评估判决** | 瞬间 | 神根据收集的数据做出判决 | — |
| **收割** | 数十年-数百年 | 征服/毁灭/改造 | 神战或文明的毁灭 |

### 6A.2 判决类型

| 判决 | 条件 | 结果 | 案例 |
|------|------|------|------|
| **清理** | 位面无资源价值，或已被污染 | 位面被彻底毁灭 | 某些已毁的低等位面 |
| **奴役** | 位面有可用的资源和劳动力 | 文明被征服，资源被提取，人口成为神的"资产" | 多个被征服的高级位面 |
| **改造** | 位面有特殊价值，需要重新塑造成神的领地 | 位面法则被重写，成为神的延伸 | 神的"神国"大多是这样创造的 |
| **收割失败** | 位面的反抗超出预期 | 神可能被击退——但这种情况极少发生 | 奥拉诺斯曾一度让神付出代价，但最终仍被毁灭 |

### 6A.3 蓝星的特殊性

蓝星在诸神的评估中有几个独特之处，这些独特性决定了故事中诸神的行为模式：

1. **人口规模前所未有**：七十亿——诸神从未面对过如此庞大的人类群体。这既让神兴奋（对玛尔、瑟瑞昂而言），也让行动变得复杂
2. **科技文明的不可预测性**：诸神熟悉魔法文明，但科技文明有完全不同的逻辑。核武器、信息网络、工业能力——这些都是诸神在历史上没有遭遇过的
3. **李梅林的隐藏干预**：李梅林在蓝星的5年间完成了神无法察觉的布局。他对神的评估机制了如指掌，并设计了应对策略
4. **神的竞争**：多个神同时对蓝星有兴趣，他们之间的竞争延缓了全面降临

### 6A.4 为什么神不立刻降临？

这是读者必然会问的问题，答案是多层的：

1. **能量成本**：神的降临需要穿越位面壁垒，消耗巨大能量。他们在评估期会尽量节省能量
2. **信息收集**：神对蓝星的了解还很有限。他们需要更多时间通过神眷者收集信息
3. **神的博弈**：没有神愿意第一个全力降临——因为那会让自己在面对其他神时处于能量不足的状态
4. **李梅林的干扰**：李梅林刻意制造了一个"蓝星正在自我毁灭"的假象——大量的裂隙被解释为位面自身的不稳定，而非人为。这让神放慢了节奏
5. **评估期的不可跳过性**：神的评估机制是刻在他们法则中的——他们不能违反自己的法则

## 七、世界观创作规则

1. **世界观的揭示必须是渐进的**：不要一次性全部告知读者
2. **男主知道的世界观远多于读者**：他的视角是全局的，但书中极少使用他的视角
3. **普通人的视角是混乱和错误的**：他们对末世的理解是片面的、恐怖的
4. **职业者的视角是逐步深入的**：随着力量增长，他们对世界真相的理解逐步加深
5. **神眷者的视角是扭曲的**：他们被信仰扭曲了认知
6. **真相在群像视角的交汇中逐步浮现**：没有一个角色能看清全貌
7. **禁用"系统"解释任何现象**：力量体系不是游戏机制，是宇宙法则的结果
8. **保持"神"的不可知感**：即使在揭示诸神真面目时，也要保留某种超出人类理解的气质
9. **末世的一切代价都要真实呈现**：不美化，不为男主辩护——让读者自己判断
10. **科学维度永远存在**：蓝星的科学家会持续用科学方法研究末世——他们的发现是揭示世界观的重要渠道
